**Emergency Services Directory** is a responsive web application that provides quick access to essential emergency services in Bangladesh. Users can call, copy numbers, and track call history in an interactive interface.

---

## üìä Project Overview

**Objective:**  
Provide citizens a convenient way to access emergency services, including police, ambulance, fire service, and more.

**Target Audience:**  
Residents of Bangladesh and users needing fast access to emergency contacts.

**Key Features:**  
- Quick access to multiple emergency services.  
- Copy phone numbers to clipboard.  
- Track call history in real-time.  
- Like/favorite services using a heart icon.  
- Coin-based call functionality for gamification.  
- Fully responsive design for desktop and mobile.  

**Tech Stack:**  
- HTML, CSS, JavaScript  
- TailwindCSS + DaisyUI for styling  
- FontAwesome icons  
- Google Fonts (Hind Madurai, Inter)  

---

## üîë Key Features

### 1. Interactive Cards
- Each emergency service has a card with name, description, and phone number.  
- Users can like/favorite a service by toggling the heart icon.  

### 2. Call Functionality
- Click the ‚ÄúCall‚Äù button to simulate calling the service.  
- Calling reduces a coin balance by 20 coins per call.  
- Alerts notify users of the call action.  

### 3. Copy Phone Numbers
- Copy service numbers to clipboard with the ‚ÄúCopy‚Äù button.  
- Copy counter keeps track of the total copies.  

### 4. Call History
- Right panel displays call history with service name, number, and timestamp.  
- Clear button allows users to reset call history.  

### 5. Coin & Love Counters
- Coins show remaining balance for calls.  
- Love counter tracks how many services the user has favorited.  

### 6. Responsive UI
- Designed with TailwindCSS and DaisyUI for desktop and mobile layouts.  
- Smooth, user-friendly interface with interactive buttons.
